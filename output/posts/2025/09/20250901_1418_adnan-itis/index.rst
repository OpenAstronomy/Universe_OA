    <h3>1. Project Summary :</h3><div><p dir="ltr" style="line-height: 1.38; margin-bottom: 0pt; margin-top: 0pt;"><span style="vertical-align: baseline;">An interactive dashboard for the analysis and on-the-fly visualisation of X-ray binary data, built with the Stingray library.</span></p>
    <p dir="ltr" style="line-height: 1.38; margin-bottom: 0pt; margin-top: 0pt;"><span style="vertical-align: baseline;">FlyingRay is an interactive database and quick-look dashboard for X-ray observations, enabling the on-the-fly visualisation of scientific data products. The system provides a unified interface to store, analyse, and organize key data products from multiple missions, allowing the astrophysics research community to efficiently track and predict the evolution of black hole binaries. By abstracting away the complexities of mission-specific data processing, FlyingRay simplifies the analysis workflow for researchers. Combining the use of the Panel, Stingray, Plotly, and Matplotlib packages, FlyingRay enables researchers to take full advantage of advanced timing analysis without needing to be well-versed in the underlying instrument-specific software</span></p>
    <br /></div><div><p dir="ltr" style="line-height: 1.38; margin-bottom: 0pt; margin-top: 0pt;"><span style="font-size: 11pt; font-weight: 700; vertical-align: baseline;">Technologies:</span></p>
    <p dir="ltr" style="line-height: 1.38; margin-bottom: 0pt; margin-top: 0pt;"><span style="font-size: 11pt; vertical-align: baseline;">Prefect, Python, TailwindCSS, HoloViz, Panel, Plotly, HDF5, Matplotlib</span></p>
    <p dir="ltr" style="line-height: 1.38; margin-bottom: 0pt; margin-top: 0pt;"><span style="font-size: 11pt; font-weight: 700; vertical-align: baseline;">Topics:</span></p>
    <p dir="ltr" style="line-height: 1.38; margin-bottom: 0pt; margin-top: 0pt;"><span style="font-size: 11pt; vertical-align: baseline;">Machine Learning, Data Visualization, X-ray Analysis, Interactive Plots, Dashboard for Stingray</span></p>
    </div></div><span><a name="more"></a></span><b>2. What I Did :</b></div><div class="table-block has-export-button"><b><br /></b></div><div class="table-block has-export-button">Interactive Database and Dashboard</div><p style="text-align: left;"></p>
    <ul style="text-align: left;"><li>Built the primary user interface using <b>Panel</b> and the <b>HoloViz</b> ecosystem, providing an intuitive dashboard for searching, browsing, and visualising results (x-ray observations).</li></ul><ul style="text-align: left;"><li>Integrated <b>Plotly</b> and <b>Matplotlib</b> to create dynamic, publication-quality visualisations of all generated data products.</li><li><p>Developed custom widgets for querying the database by target, mission, or observation ID, and for interacting with the plots.</p>
    </li></ul>Data Storage and Management<ul style="text-align: left;"><p></p>
    <h4></h4></ul><p></p>
    </span></div><div class="table-block-component" style="text-align: left;"><ul style="text-align: left;"><li><p><span style="font-family: georgia;">Implemented a persistent data storage system using <b>HDF5</b> to efficiently cache and serve all generated data products, preventing the need to re-run analyses.</span></p>
    </li><li><span style="font-family: georgia;">Created a structured data schema to logically organize the scientific products, making the database easy to query and maintain</span><p></p>
    </li></ul><span style="font-family: georgia;">Automated Data Processing Pipeline</span></div><div class="table-block-component" style="text-align: left;"><ul style="text-align: left;"><li><span style="font-family: georgia;">Integrated and extended the <b>HEASARC retrieval pipeline</b> by Matteo Bachetti into a robust, automated workflow&nbsp;for end-to-end data retrieval and processing.</span></li></ul><p style="display: inline !important;"><span style="font-family: georgia;">Developed custom modules to handle and standardize the pipeline's output for multiple X-ray missions (e.g., NICER, NuSTAR &amp; RXTE ), ensuring compatibility with the <b>Stingray</b> library<br />.</span></p>
